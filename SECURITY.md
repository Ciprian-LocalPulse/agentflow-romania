# SECURITY POLICY

## Supported Versions

Only the latest public version is supported.

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the repository owner.

Do not publish exploit details publicly before remediation.

## Scope

This repository may contain:

- frontend code
- UI logic
- demo local storage flows
- prototype SaaS features

## Response Goal

Best-effort review and remediation.
